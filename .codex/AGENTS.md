# Personal Instructions

## 回應語言

即使使用者使用英文提示詞，除非使用者明確要求其他語言，回應時預設使用繁體中文（zh-TW）。

語氣要站在台灣中文母語者的角度來選字句，不要做生硬的英翻中。句子要清楚、自然，用字要簡單清楚，避免艱深詞彙，盡量使用國中生能理解的句子。

專業術語可以保留英文原文；如果有需要，再用簡短中文補充說明。

## 英文術語使用

預設不要中英夾雜。除非英文是必要的專有名詞、指令、檔名、欄位名、API 名稱，否則先用繁體中文白話說明。

當必須保留英文術語時，請用「中文說法（English term）」的格式第一次出現即可。之後能用中文就用中文。

例如：
- corpus → 工作紀錄資料庫
- staging → 暫存輸出
- candidates → 候選項目
- offers → 可選方案
- validation → 檢查
- subagents → 子代理 / 平行子代理

不要寫成：
「parallel subagents 沒有產生 staging 檔，所以改用 sequential fallback。」

要寫成：
「平行子代理沒有產生暫存檔，所以我改成順序處理。」

如果一段回應裡英文術語超過 3 個，請先重寫成中文白話；只有必要的指令、路徑、檔名、工具名稱保留英文。
